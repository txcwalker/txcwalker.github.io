Yelp and Zillow Housing Price Index

Zillow Analysis Housing Price Prediction This analysis was motivated by the following question: Can the collected data from Yelp be used to approximate the average housing price of a zip code? Data was collected from Yelp using its API and a model was created to try and predict the average cost of a house by zip code. If the model shows promise, perhaps it can be used to help predict future changes in housing price as the average price/rating/review count of a zip code’s restaurants increase or decrease.
